
# 🌌 Interactive 3D Solar System Model

A visually engaging and interactive **3D Solar System simulation** built using pure HTML, CSS, and JavaScript.  
Experience planetary motion, real-time interactions, and detailed astronomical data in a modern web interface.

🔗 Live Demo: https://interactive-3-d-solar-system-model.vercel

---

## 🚀 Features

### 🪐 3D Visualization
- Realistic 3D solar system using CSS perspective & transforms  
- Proportional planetary sizes and orbital distances  
- Smooth orbital animations with adjustable speed  
- Dynamic lighting and depth effects  

### 🎮 Interactive Controls
- Drag to rotate the system  
- Scroll to zoom in/out  
- Click planets to view details  
- Toggle orbits, labels, and rotation  
- Reset view to default  

### 📊 Information Panel
- Planet data inspired by NASA datasets  
- Real-time statistics and measurements  
- Clean UI for displaying planet details  
- Fully responsive across devices  

### 🌟 Visual Effects
- Animated starry background  
- Glowing sun with particle effects  
- Planet-specific textures and colors  
- Saturn rings rendered in 3D  
- Smooth transitions and hover effects  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)  
- **3D Effects:** CSS Transforms & Perspective  
- **Icons:** Font Awesome  
- **Fonts:** Google Fonts (Orbitron, Exo 2)

## ⚙️ How It Works

- Uses **CSS 3D transforms** to simulate planetary orbits  
- JavaScript controls animation speed, rotation, and interaction  
- Event listeners handle user input (drag, click, zoom)  
- Data-driven UI dynamically updates planet information  

---

## 🎯 Key Highlights

- No external 3D libraries (like Three.js) used  
- Pure CSS-based 3D rendering  
- Smooth performance with optimized animations  
- Clean separation of structure, styling, and logic
